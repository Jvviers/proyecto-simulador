Para su utilización, dado que la interfaz no es muy amigable, es importante seguir los siguientes pasos:

Pasos:

- Elegir nombre del proceso
- Tiempo de llegada
- Burst
- Memoria

- Es importante quitar el tick de prioridad cada vez que se cree un proceso nuevo en caso de que no queramos utilizar la planificación por multi nivel.
- Si está elegido el SJF no es necesario que modifiquemos el quantum, debido a que el programa se encarga de no considerar el quantum.
- Si está elegido el RR ahí si es necesario elegir quantum.

Por otro lado, si deseamos probar la planificación por prioridad, creamos un proceso considerando el check y seleccionando si queremos el proceso con una prioridad
alta o baja. Ahí no importa elegir algoritmo de planificación ya que, en el algoritmo, ya hace bien el proceso de no considerarlo.

- Luego, seleccionar el quantum que deseemos. 

Se comprende que la interfaz no es muy amigable, debido a que nos enfocamos en el algoritmo como tal.

Se solicita que en caso de tener problemas con Electrón, instalar las dependencias con: 

pnpm install 
pnpm approve-builds

y ejecutar normal con: 

npm start
